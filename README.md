# AURA - AI-Powered Unified Repository Application

This project consists of a React frontend and a FastAPI backend.

## Prerequisites

- Python 3.7+ with pip
- Node.js with npm

## Backend Setup

1.  **Navigate to the root directory:**
    ```bash
    cd c:\Users\revan\Downloads\AURA
    ```

2.  **Install Python dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the backend server:**
    ```bash
    python backend_complete.py
    ```
    The backend will be running at `http://localhost:8000`.

## Frontend Setup

1.  **Navigate to the frontend directory:**
    ```bash
    cd c:\Users\revan\Downloads\AURA\aura-frontend
    ```

2.  **Install npm dependencies:**
    ```bash
    npm install
    ```

3.  **Run the frontend development server:**
    ```bash
    npm run dev
    ```
    The frontend will be running at `http://localhost:5173`.

## Running the Application

After starting both the backend and the frontend, you can access the application in your browser at `http://localhost:5173`.
