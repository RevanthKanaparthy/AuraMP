# How to Run

## Initial Setup

Before running the application for the first time, you need to install the required Python dependencies. Open a terminal in the project root and run:

```bash
pip install -r requirements.txt
```

## Running the Backend

To run the backend server, execute the following command:

```bash
start_backend.bat
```

## Running the Tests

To run the tests, you can use `pytest`.

To run all the tests, run the following command:

```bash
pytest
```

To run a specific test file, you can run the following command:

```bash
pytest <test_file_name>.py
```
